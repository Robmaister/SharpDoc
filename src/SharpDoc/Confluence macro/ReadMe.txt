To compile the macro surce code :

- dowload atlassian Confluence SDk
- launch the 'atlas-compile' in the macro src directory (it generate a 'target' directory with the .jar file)



To add the macro to a confluenec server :
(need to have admin rights)

- go to admin menu
- go to 'add-on' or 'plug-in' section
- click on 'download plug-in' and select the .jar file
